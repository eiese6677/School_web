# 자리 바꾸기 웹 애플리케이션

## 설명

이 프로젝트는 교실의 학생 자리 배치를 도와주는 간단한 웹 애플리케이션입니다. 학생 명단을 기반으로 자리를 무작위로 섞거나, 특정 학생의 자리를 직접 바꿀 수 있습니다. 자리 배치가 완료되면 현재 상태를 서버에 저장하여 다음에 다시 불러올 수 있습니다.

## 주요 기능

- **학생 명단 관리**: `data.json` 파일에 저장된 학생 명단을 불러와 화면에 표시합니다.
- **자리 섞기**: "자리바꾸기" 버튼을 클릭하여 학생들의 자리를 무작위로 섞습니다.
- **학생 자리 교환**: 두 학생의 이름을 차례로 클릭하여 서로의 자리를 바꿀 수 있습니다.
- **자리 고정**: 학생의 이름을 두 번 클릭하여 자리를 고정하거나 고정을 해제할 수 있습니다. 고정된 학생은 자리 섞기 시에도 위치가 변경되지 않습니다.
- **자리 배치 저장하기**: "저장" 버튼을 클릭하여 현재 자리 배치를 서버에 저장하고, 페이지가 로드될 때 마지막 자리 배치를 자동으로 불러옵니다.
- **자리 배치 불러오기**: 이전 기록 중 원하는 저장기록을 불러올 수 있습니다.
- **중복 방지**: 되도록이면 옆자리에 이 전과 같은 친구가 앉지 않게 합니다. (끄기 가능)
- **거리 설정**: 친구들 간의 대략적인 거리를 지정해 가까이 또는 멀리 앉힐 수 있습니다.

## 파일 구조

```
.
├── app.py              # Flask 웹 서버
├── data.json           # 학생 명단 및 자리 배치 데이터
├── static
│   ├── script.js       # 프론트엔드 JavaScript 코드
│   └── style.css       # CSS 스타일시트
└── templates
    └── index.html      # 메인 HTML 파일
```
