# 자리 바꾸기 웹 애플리케이션

## 설명

이 프로젝트는 교실의 학생 자리 배치를 도와주는 간단한 웹 애플리케이션입니다.  
학생 명단을 기반으로 자리를 무작위로 섞거나, 특정 학생의 자리를 직접 바꿀 수 있습니다.  
자리 배치가 완료되면 서버에 저장하고, 저장된 기록은 언제든 다시 불러올 수 있습니다.

> 이 프로젝트는 AI의 도움을 받아 제작되었습니다.

---

## 주요 기능

- **학생 명단 관리**: JSON 파일에서 학생 명단을 불러와 화면에 표시
- **자리 섞기**: 버튼 클릭을 통해 자리를 무작위로 섞기
- **학생 자리 교환**: 서로 다른 이름을 클릭하여 서로 위치 바꾸기
- **자리 고정**: 특정 학생은 자리를 고정시켜 섞을 때 위치 유지 (같은 자리를 2번 연속 클릭하면 활성화/비활성화 전환 가능)
- **자리 배치 저장**: 현재 자리 상태를 서버에 저장
- **기록 불러오기**: 이전에 저장한 배치 기록을 선택해서 다시 불러오기
- **중복 방지**: 이전과 같은 친구가 옆자리에 앉는 것을 되도록 피함 (옵션 설정 가능)
- **거리 계산 기능**: 학생 간 거리 고려한 배치 가능

## 설치 및 실행 방법

### 1. Python 및 Flask 설치

```bash
pip install flask
```

### 2. 서버 실행

```bash
python main.py
```

### 3. 브라우저 접속

브라우저에서 아래 링크 또는 bash에 나오는 링크로 접속
http://localhost:5000

---

## API

### GET `/load`

- 학생 명단과 자리 기록 데이터를 반환

### POST `/save`

- 자리 배치를 서버에 저장
- 요청 형식(JSON):

```json
{
  "label": "label1",
  "last": [0, 1, 2, 3, ...]
}
```
